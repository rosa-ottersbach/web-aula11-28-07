O desenvolvimento web se trata da criação de um software, seja ele um sistema, um app, um portal ou um site, que está conectado a uma rede, podendo ser internet ou intranet (uma rede interna). Softwares web são desenvolvidos para realizar operações em tempo real.

Hoje, a maior parcela do que é desenvolvido é feito para web, permitindo conectividade, integração entre diferentes softwares e ações com resultados instantâneos.
